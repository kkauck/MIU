About Game Tracker

This application was deisgned to help both the casual and hardcore gamer keep track of all the games that they own. It is an easy app to add and navigate to and find the games you have collected. Everything is being worked on to make sure you as a user have an easy time both adding and finding exactly what you want from this app.
